# vw适配移动端
> Built by Parcel, React

在看文大漠老师的 [分享手淘过年项目中采用到的前端技术](https://www.w3cplus.com/css/taobao-2018-year.html)后学习demo，svg资源来自iconfont以及图片资源使用了手淘中的资源。

## Installation
```
$ npm install
$ npm run dev
```
